# Pet Crypto

A Flutter pet project for browsing cryptocurrency data.

This project was created to demonstrate my approach to Flutter application architecture,
state management, networking, error handling, authentication,
local storage, localization, and testing.

## Specify using the --dart-define option:
- AUTH_URL - Base URL of the authentication API (https://dummyjson.com)
- API_URL - Base URL of the cryptocurrency API (https://pro-api.coinmarketcap.com)
- API_KEY - Cryptocurrency API key
